# 로그인 API 요청 규격 (프론트 기대치)

- `POST /auth/login`
- Headers: `Content-Type: application/json`
- Body:
  ```json
  {
    "email": "user@example.com",
    "password": "P@ssw0rd1"
  }
  ```
- Response (성공):
  ```json
  {
    "ok": true,
    "message": "login success"
  }
  ```
- 비고:
  - 쿠키 기반 JWT가 `access_token`에 설정되어 이후 `/auth/me` 호출로 세션을 확인합니다.
  - 이메일/비밀번호 불일치 또는 계정 차단 시 401 응답을 예상합니다.
